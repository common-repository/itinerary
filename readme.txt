== Description ==

Create an itinerary like functionality for your WordPress site

= Compatibility =

WP >= v3.5

== Installation ==

1. Copy the `itinerary` folder into your `wp-content/plugins` folder
2. Activate the Itinerary plugin via the plugins admin page
